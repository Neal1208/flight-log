# 学员飞行日志 - 需求规格说明

## 项目概述

为飞行学员打造的手机端 Web 应用，用于记录每日飞行训练情况。支持多用户使用，提供日历选择、课程记录、未飞行原因标记、语音输入和数据导出功能。

## 用户角色

- **学员**：核心用户，记录个人飞行日志
- 管理员功能留待后续迭代

## 功能需求

### F1. 用户系统

- 手机号注册/登录
- 登录状态持久化（关闭浏览器后保持登录）
- 退出登录

### F2. 日历日期选择

- 以月历形式展示，格式为「YYYY年 M月 D日」
- 支持左右箭头切换月份
- 点击日期选中，选中日期高亮显示
- 默认选中当天

### F3. 飞行大纲课程记录

- 用户可自由编辑，每行包含：
  - 科目名称（文本输入）
  - 飞行时长（数字输入，单位：小时）
- 支持添加多行课程
- 支持删除已添加的行

### F4. 未飞行原因标记

- 以标签选项形式呈现，点击选中/取消：
  1. 天气原因 — 可添加备注
  2. 运行限制 — 可添加备注
  3. 身体原因 — 可添加备注
  4. 个人原因 — 可添加备注
  5. 英语限制 — 可添加备注
  6. 转场限制 — 可添加备注
  7. 公司原因 — 可添加备注
- 支持多选
- 每个选项选中后可输入备注文字

### F5. 语音输入

- 点击语音按钮开始录音
- 通过浏览器语音识别自动转为文字
- 文字自动填入当前焦点的输入框

### F6. 数据保存

- 所有填写内容保存到云端（LeanCloud）
- 按用户 + 日期唯一存储
- 支持修改已有记录

### F7. 报表导出

- 选择日期区间（开始日期 ~ 结束日期）
- 预览区间内数据
- 导出为 Excel 文件（.xlsx）
- 导出为 PPT 文件（.pptx）
- 导出内容包含：日期、课程、时长、未飞行原因、备注

## 非功能需求

- **移动端优先**：以手机屏幕为主要设计目标
- **响应式设计**：适配不同屏幕尺寸
- **简约风格**：UI 简洁清晰，操作直观
- **免费部署**：使用免费云服务，无服务器成本
- **离线提示**：网络异常时给出友好提示
