# 学员飞行日志 - UI 设计规范

## 设计原则

- **极简主义**：减少视觉噪音，每个页面只呈现核心信息
- **移动端优先**：以 375px 宽度为基准设计，向上兼容
- **高可读性**：充足留白，清晰信息层级
- **即时反馈**：按钮点击有视觉响应，操作结果有明确提示

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色（深色） | #1a1a2e | 顶栏、主按钮、选中状态 |
| 强调色（蓝色） | #4a90d9 | 链接、语音按钮、可操作元素 |
| 成功色（绿色） | #217346 | Excel 导出按钮 |
| 警示色（红色） | #d24726 | PPT 导出按钮、删除操作 |
| 背景色 | #f5f5f5 | 页面底色 |
| 卡片色 | #ffffff | 内容卡片背景 |
| 边框色 | #e0e0e0 | 输入框、卡片边框 |
| 文字主色 | #1a1a2e | 标题、正文 |
| 文字次色 | #666666 | 说明文字 |
| 文字弱色 | #999999 | 占位符、辅助信息 |

## 字体

- 系统默认字体栈：`-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Microsoft YaHei", sans-serif`
- 标题：16-22px，font-weight 600-700
- 正文：13-15px，font-weight 400
- 辅助文字：11-12px

## 间距系统

- 页面边距：16px
- 卡片内边距：14-16px
- 卡片间距：10px
- 元素间距：8-12px
- 组件圆角：8-12px
- 标签圆角：16-20px（胶囊形）

## 组件规范

### 按钮
- 主按钮：深色背景 #1a1a2e，白色文字，圆角 8px，高度 44-48px
- 次按钮：浅色边框，透明背景
- 语音按钮：圆形 42px，蓝色背景，麦克风图标

### 输入框
- 边框 1px solid #e0e0e0
- 圆角 8px
- 内边距 10-12px
- 聚焦时边框变为 #4a90d9

### 标签（未飞行原因）
- 胶囊形，padding 5px 10px
- 默认：浅灰背景 + 灰色边框
- 选中：深色背景 #1a1a2e + 白色文字
- 字号 11px

### 日历
- 7 列网格布局
- 选中日期：深色圆形背景
- 当天：加粗显示
- 月份切换箭头：左右两侧

### 底部导航
- 三个标签：日志 / 报表 / 我的
- 当前页高亮，其余灰色
- 高度 48px
